This post in the Tcl Master Series is dedicated to discuss various types of loops and their operations in Tcl.

Though there exists several kinds of looping systems available in Tcl. Primarily, for, foreach and while are very popular in Tcl. Each of them can be defined in the following ways

  1. for Loop 
     set N 100;
 for {set i 1} {$i < $N} {incr i 1} {
     puts "The Value of i = $i";
 }
  2. foreach Loop 
     set Variable_Vector "1 2 3 4 5 6 7 8 9";

 foreach i $Variable_Vector {
     puts "The Value of i = $i";
 }
  3. while Loop 
     set i 0;
 set N 100;
 while {$i < $N} {
     puts "The Value of i = $i";
     set i [expr ($i + 1)];
 }

    or

     set i 0;
 set N 100;
 while {$i < $N} {
     puts "The Value of i = $i";
     incr i; # incr is equivalent to (set i [expr ($i + 1)])
 }
